# White Noise 项目功能和页面结构说明文档

## 一、项目概述

**项目名称**：white_noise（白噪音）

**项目定位**：一款集环境音播放、冥想引导、DIY 音效组合和趣味互动于一体的**心理舒缓与助眠应用**。通过多种放松方式帮助用户减压、助眠、提升专注力。

**技术栈**：Flutter（跨平台移动应用框架）

---

## 二、应用整体架构

### 2.1 导航结构

应用采用**底部导航栏（Bottom Navigation Bar）**作为主导航，包含四个核心功能模块：

| 导航项 | 图标 | 功能说明 |
|--------|------|----------|
| **睡眠** | 月亮 + Z 字符 | 定时睡眠，播放环境音助眠 |
| **DIY** | 调节器/均衡器 | 自定义组合多种音效 |
| **冥想** | 盘腿坐姿人物 | 冥想引导课程 |
| **木鱼** | 木鱼形状 | 电子木鱼，功德积累趣味互动 |

### 2.2 页面层级关系

```
应用入口
    └── 主框架（底部导航）
            ├── 睡眠页（定时睡眠）
            ├── DIY 页（DIY 白噪音）
            ├── 冥想页（放松冥想）
            └── 木鱼页（功德）
```

---

## 三、各页面功能详解

### 3.1 睡眠页（定时睡眠）

**页面标题**：定时睡眠

**核心功能**：播放环境音辅助睡眠，支持倒计时自动停止。

**功能说明**：环境音播放会在倒计时结束后自动停止。

**页面结构**：

| 区域 | 内容 |
|------|------|
| 顶部 | 标题「定时睡眠」+ 功能说明文案 |
| 中部 | 定时器显示（如 15:00），可点击调整时长 |
| 分类导航 | 水平滚动的音效分类标签：事物、雨、动物、城市、场所、交通、自然、白噪 |
| 音效网格 | 当前分类下的环境音选项，以网格形式展示 |

**「事物」分类下的音效示例**：
- 颂钵、吹风机、雨刮器、风铃、洗衣机
- 黑胶唱片效果、打字机、正在调谐的收音机、幻灯机
- 沸水、纸、摩尔斯电码、键盘、烘干机、时钟、吊扇等

**交互逻辑**：
- 选择分类 → 展示该分类下的音效列表
- 选择音效 → 开始播放
- 设置定时器 → 倒计时结束后自动停止播放

---

### 3.2 DIY 页（DIY 白噪音）

**页面标题**：DIY 白噪音

**核心功能**：选择多个音效并独立调节音量，组合成个性化的白噪音。

**功能说明**：选择多个音频，更改音频中间的音量键可配置自己的组合音效。

**页面结构**：

| 区域 | 内容 |
|------|------|
| 顶部 | 标题「DIY 白噪音」+ 功能说明 |
| 控制区 | 「小 83 音效」按钮、「分类」「播放」切换按钮 |
| 音效列表 | 分类标题 + 网格布局的音效卡片 |

**音效卡片结构**（每行约 3 个）：
- 选择按钮（⌘ 符号）
- 音量滑块（可独立调节该音效音量）
- 音效名称

**音效示例**：幻灯机、吹风机、雨刮器、风铃、洗衣机、黑胶唱片效果等。

**交互逻辑**：
- 点击选择按钮 → 添加/移除该音效到组合
- 拖动音量滑块 → 调节该音效在组合中的音量
- 点击「播放」→ 播放当前组合音效

---

### 3.3 冥想页（放松冥想）

**页面标题**：放松冥想

**核心功能**：提供冥想引导课程，配合环境音进行冥想练习。

**功能说明**：环境音播放会在倒计时结束后自动停止。

**页面结构**：

| 区域 | 内容 |
|------|------|
| 顶部 | 标题「放松冥想」+ 功能说明 |
| 内容区 | 垂直滚动的冥想课程列表（卡片式布局） |

**课程卡片结构**：
- 左侧：方形缩略图（代表冥想主题）
- 右侧：主标题、副标题、描述、时长

**冥想课程示例**：

| 主标题 | 副标题 | 描述 | 时长 |
|--------|--------|------|------|
| 让生活慢下脚步 | 10分钟冥想引导 | 在自然中，感受呼吸的力量 | 10分钟 |
| 提升你的内在安全感 | 10分钟冥想引导 | 提升你的内在安全感 | 10分钟 |
| 我生活的太严肃，忘记了对一朵花微笑 | 10分钟冥想引导 | — | 10分钟 |
| 在冥想中遇见真实的自己 | 接纳自我 | 每天10分钟，与内心对话 | 9分钟 |
| 修复亲密关系中的不安全感 | 安全感 | — | 9分钟 |

**交互逻辑**：
- 浏览课程列表 → 选择感兴趣的冥想课程
- 点击课程卡片 → 进入播放/引导页面

---

### 3.4 木鱼页（功德）

**页面标题**：功德

**核心功能**：通过敲击电子木鱼实现心理舒缓，以趣味方式减压。

**功能说明**：通过敲电子木鱼功能实现心理舒缓。

**页面结构**：

| 区域 | 内容 |
|------|------|
| 顶部 | 标题「功德」+ 功能说明 |
| 中部 | 功德数值大号显示（如「7」） |
| 中部 | 「累积功德: 7」文字说明 |
| 中部 | 动态提示「功德+1」（敲击时短暂显示） |
| 底部 | 电子木鱼图标（主要互动元素） |

**交互逻辑**：
- 点击木鱼图标 → 敲击一次，功德 +1
- 功德数值实时更新
- 「功德+1」提示短暂显示后消失

## 四、UI 设计要点

### 4.1 视觉风格
- 背景：白色/浅色为主，界面简洁
- 图标：黑色方形/圆形图标，风格统一
- 分类选中：蓝色指示线高亮

### 4.2 通用组件
- **底部导航栏**：四宫格，图标 + 文字
- **音效卡片**：图标 + 名称，可选音量滑块（DIY 页）
- **分类标签**：水平滚动，支持多分类切换

### 4.3 音效分类体系
- 事物、雨、动物、城市、场所、交通、自然、白噪等
